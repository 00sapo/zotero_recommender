# zotero_recommender

## Installation

`npm install -g zotero_recommender`

## Usage

`zotero_recommender --help`

In the UI, you can move with `j` and `k`, select with `enter`, exit with `q`, `esc` or
`ctrl+c`. You can also search with `/` .

## Development

1. Install Zotero
2. Install Node.js
3. `corepack enable yarn`
4. `yarn install`
5. `yarn start`
